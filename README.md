# vue-content-wrapper

> A Vue component for creating a section container with a inner div content element, capable of mapping attributes and listeners between both

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
